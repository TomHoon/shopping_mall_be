# 최초 서버 기동 가이드 (테스트 데이터 주입) 사용안함❌. 볼필요없음
> 서버 기동시 테이블 생성 후 데이터 주입이 필요하여 아래와 같은 설정이 필요함
---

1. application-local.yml에서의 data init never로 설정, hibernate create-drop으로 설정
```dbn-psql
spring:
  sql:
    init:
#      mode: always
       mode: never
#       mode: embedded


... (생략)

  jpa:
    hibernate:
#      ddl-auto: update
      ddl-auto: create-drop
```
2. 서버 기동 후 종료
3. 다시 아래 설정으로 원복
```dbn-psql
spring:
  sql:
    init:
      mode: always
#       mode: never
#       mode: embedded

... (생략)

  jpa:
    hibernate:
      ddl-auto: update
#      ddl-auto: create-drop
```
