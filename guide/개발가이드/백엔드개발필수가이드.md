# 백엔드 개발 필수 가이드

이 문서는 프로젝트에서 **모든 백엔드 개발자가 반드시 지켜야 하는 개발 규칙**을 정의한다.
해당 규칙은 코드 스타일 통일, 유지보수성 확보, 팀 협업 효율 향상을 목적으로 한다.

---

# 1. Swagger 문서 작성 필수

모든 API는 **Swagger 문서 작성을 필수로 한다.**

목적

```
API 명세 자동 문서화
프론트엔드 협업 지원
테스트 편의성 확보
```

필수 항목

```
API 설명
Request 파라미터 설명
Response 데이터 설명
```

예시

```
@Operation(summary = "상품 상세 조회", description = "상품 ID로 상품 상세 정보를 조회한다.")
```

---

# 2. JPA / MyBatis 사용 규칙

본 프로젝트에서는 **JPA와 MyBatis를 함께 사용한다.**

규칙

```
1. JPA
   - 테이블 생성
   - 간단한 CRUD

2. MyBatis
   - 모든 조회 Query
   - 복잡한 비즈니스 Query
```

연관관계 규칙

```
허용
@OneToMany

금지
@ManyToOne
```

이유

```
불필요한 Lazy Loading 방지
N+1 문제 방지
Query 명확성 확보
```

---

# 3. 코드 생성 순서

개발 시 **아래 순서를 반드시 지켜야 한다.**

```
mapper.xml
→ mapper.java (interface)
→ service.java
→ controller.java
```

이 순서를 따르는 이유

```
Query 기준 개발
비즈니스 로직 흐름 명확화
개발 표준 통일
```

---

# 4. Mock 응답 데이터 작성

모든 API 작업 완료 후 **Mock 응답 데이터를 반드시 작성해야 한다.**

위치

```
mock/
```

파일명 규칙

```
endpoint 명 사용
```

예시

```
GET /api/products/{id}

파일명
product-detail.json
```

목적

```
프론트엔드 개발 지원
API 테스트 데이터 제공
문서화
```

---

# 5. 공통 응답 처리

본 프로젝트는 **공통 응답 구조를 사용한다.**

개발자는 **Controller에서 직접 공통 응답을 지정할 필요 없다.**

응답 구조는

```
ApiResponseAdvice
```

에서 자동으로 처리된다.

응답 예

```
{
  "data": {},
  "message": "success",
  "statusCode": 200
}
```

---

# 6. 테스트 코드 작성 필수

모든 작업 완료 후 **테스트 코드 작성은 필수이다.**

작성 대상

```
Controller 테스트
Service 테스트
```

목적

```
API 동작 검증
리팩토링 안정성 확보
버그 방지
```

테스트 예

```
MockMvc
Service Unit Test
```

---

# 7. 주석 작성 규칙

모든 비즈니스 로직에는 **주석을 반드시 작성해야 한다.**

목적

```
신규 개발자 이해 지원
비즈니스 흐름 명확화
유지보수성 확보
```

주석 기준

```
코드를 처음 보는 사람이
비즈니스 로직 흐름을 한 눈에 이해할 수 있어야 한다.
```

---

# 8. 추가 가이드 문서 작성

비즈니스 로직이 복잡하거나
추가 설명이 필요한 경우 **별도 가이드 문서를 작성해야 한다.**

위치

```
guide/개발가이드/도메인/
```

파일명 규칙

```
~데이터추가가이드.md
```

예시

```
상품데이터추가가이드.md
주문데이터추가가이드.md
정산데이터추가가이드.md
```

문서 내용

```
데이터 생성 방법
테이블 관계 설명
비즈니스 정책 설명
```

---

# 정리

모든 백엔드 개발자는 아래 규칙을 반드시 지켜야 한다.

```
Swagger 문서 작성
JPA / MyBatis 사용 규칙 준수
코드 생성 순서 준수
Mock 응답 데이터 작성
공통 응답 구조 사용
테스트 코드 작성
주석 작성
가이드 문서 작성
```

이 규칙은 **프로젝트 전체 코드 품질과 협업 효율을 위해 반드시 준수해야 한다.**
